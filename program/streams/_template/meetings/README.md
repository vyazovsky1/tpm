# Meetings — {{STREAM}}

_Maintained by Meetings Agent. All changes require TPM approval._

One file per meeting summary lives in this directory — **the directory listing is the meeting
log**; there is no separate index file. Name each summary `YYYY-MM-DD-<type>.md`
(e.g. `2026-06-08-standup.md`).

Each summary follows the **common template** shared by all streams —
`skills/meeting-summary/summary-template.md` in the AITPM framework — copied here and filled in
by the Meetings Agent. Decisions are also recorded in `../decisions.md`, action items in
`../action-items.md`, and **Notes source** links the actual resource (from integration metadata).

Follow-up agenda documents are created by `/tpm:meeting-prep` and retired to `../history/`
once the next summary is written.
